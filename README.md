# 🧭 CampusCompass
### *Navigate Your Campus Journey.*

CampusCompass is an AI-assisted, Python-based recommendation system that helps students discover campus events based on their interests, previous participation, and event popularity. The goal is to make it easier for students to find opportunities that match their passions and encourage greater involvement in campus activities.

---

## 📌 Problem Statement

Many students miss valuable campus events because:

- They are unaware of upcoming events.
- Event information is scattered across different platforms.
- Students struggle to find events that match their interests.
- There is no personalized recommendation system for campus activities.
- New students often don't know which events are worth attending.

As a result, students miss opportunities to learn new skills, network with peers, and participate in extracurricular activities.

---

## 💡 Our Solution

CampusCompass addresses these challenges by providing personalized event recommendations using student interests and historical participation data.

The system helps students by:

- 🎯 Recommending events based on their interests.
- 🤝 Suggesting events using previous participation history.
- 🔥 Highlighting trending events on campus.
- 📅 Displaying upcoming events happening within the next seven days.
- 📜 Showing each student's event participation history.

This makes discovering relevant campus opportunities faster, easier, and more personalized.

## ⚙️ How It Works

1. Loads student, event, and interaction datasets.
2. Identifies the student using their roll number.
3. Displays previously attended events.
4. Accepts the student's interests.
5. Recommends matching events.
6. Uses participation history to recommend additional events.
7. Shows trending events if personalized recommendations are unavailable.
8. Displays upcoming events for the next seven days.
